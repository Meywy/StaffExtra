# StaffExtra

StaffExtra is Minecraft Plugin for Server Staff Members

## Installation



```
Clone this repo, then build it using Maven
```

## Usage

```
/staffmod <on/off> - Turns StaffMod ON/OFF
/staffmenu - Opens Staff Menu
```
